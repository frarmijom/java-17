# 🧪 Balance Algebraico – Laboratorio Visual

## 📌 Propósito
Construir un sistema interactivo que permita aprender álgebra básica, entender ecuaciones como equilibrios, visualizar transformaciones matemáticas, aplicar operaciones sin perder el equilibrio y conectar matemáticas con sistemas reales.

## 🧠 Idea Central
Una ecuación como `2x + 3 = 7` no es solo una expresión simbólica: representa un equilibrio entre dos lados.

## ⚖️ Interpretación
- Lado izquierdo: `2x + 3`
- Lado derecho: `7`

Ambos lados tienen el mismo “peso”. Resolver la ecuación significa mantener el equilibrio mientras transformas ambos lados hasta aislar la variable.

## 🎯 Objetivo del Sistema
Permitir al usuario:
1. Definir una ecuación.
2. Visualizar ambos lados.
3. Aplicar operaciones válidas.
4. Observar cómo cambia el sistema.
5. Llegar a la solución paso a paso.

## 🔹 Alcance Matemático
### Nivel 1
Ecuaciones simples: `ax + b = c`

### Nivel 2
Ecuaciones con términos en ambos lados: `2x + 3 = x + 5`

### Nivel 3
Simplificación:
- combinar términos
- mover términos entre lados

### Nivel 4
Resolución completa paso a paso.

## 🧩 Modelo Conceptual
### Entidades principales
- `Equation`
- `Expression`
- `Term`
- `Variable`
- `Constant`

### Representación conceptual
- Left: `2x + 3`
- Right: `7`

## 🔧 Operaciones del Sistema
Las operaciones deben mantener el equilibrio:
- sumar a ambos lados
- restar a ambos lados
- multiplicar ambos lados
- dividir ambos lados
- simplificar términos

## 🧠 Principio Fundamental
Toda operación debe aplicarse a ambos lados para mantener el equilibrio.

## 🎨 Visualización
El sistema representará:
- dos lados (izquierdo y derecho)
- términos como bloques
- variables como elementos diferenciados
- constantes como unidades

La inspiración visual es una balanza física.

## 🚀 MVP (Primera versión)
### Funcionalidad mínima
- ingreso de ecuación tipo `ax + b = c`
- mostrar ambos lados
- resolver paso a paso
- mostrar transformaciones

## 🔥 MVP 2 (Visual)

## 🧠 Insight Clave
Resolver ecuaciones no es manipular símbolos; es mantener un equilibrio mientras transformas el sistema.
